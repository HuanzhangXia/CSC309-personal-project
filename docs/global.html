options-DestinatedBubble

The global options that will be passed to the constructor of the DestinatedBubble.

Type:
  • Object
Properties:
Name Type Attributes Description
speed Array.<number>

An array of numbers that specifies the speed of the movement. The array should contain two numbers.
boardsize Array.<number>

An array of numbers that specifies the width and height of the moving area. The array should contain two numbers.
moveupdate number <optional>

The update interval between the movements (in ms).
moveTime number <optional>

The total time of the timed movement.
destination Array.<number> <optional>

The destination of the element.
